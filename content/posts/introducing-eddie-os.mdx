For years my site was a perfectly nice blog. Nice — but every nice blog looks like every other nice blog. So I rebuilt it as something I'd actually want to click around in: **Eddie OS**, a tiny desktop environment that boots up right in your browser.

::sketch{name=particles height=320}

## Why a whole operating system?

A personal site should feel personal. Most portfolios are a hero section, a grid of cards, and a contact form. I wanted mine to invite a bit of play — to reward poking around. A desktop metaphor does that for free: there are windows to drag, icons to open, a dock to explore, and a terminal for the curious.

:::callout{type=tip}
Drag any window by its title bar, double-click the title bar to maximise, or hit the **⤢** button to go fullscreen. Every app even has its own URL — try `/tetris`.
:::

It also scratches a nostalgic itch. I grew up dragging windows around old versions of macOS and Ubuntu, and there's something delightful about that tactile, "this is *my* machine" feeling.

## Posts that can run code

This is the fun part. Blog posts here aren't just text — they're MDX, so they can embed **live JavaScript canvases**, videos, and syntax-highlighted code right inline. That particle field up top? It's a real `<canvas>` reacting to your cursor. Here's Conway's Game of Life — click it to reseed:

## How it's built

The whole thing is a single Next.js page that hands off to a client-side window manager. State lives in a tiny Zustand store — opening an app is one call:

```tsx
// Focus an existing single-instance window, or spawn a fresh one.
openApp('blog', { props: { slug: 'introducing-eddie-os' } });

// The address bar stays in sync with whatever window is focused,
// so every view is a shareable deep link: /blog/introducing-eddie-os
```

The chrome and animations are React, Tailwind and Framer Motion. There's no heavyweight CMS anymore — posts are just Markdown/MDX files read at build time, and rich blocks (code, video, callouts, canvases) are plain React components.

::sketch{name=dvd height=180}

It's deliberately over-engineered for a personal site, and that's exactly the point. Pull up a window, open the Terminal, and have a look around.
